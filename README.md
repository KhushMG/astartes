## WH-40k Astartes Database (Unofficial)
Using this website, a user may search for a certain Astartes by name, chapter, or allegiance (loyalist or traitor). <br>

<br/>
- Next.js + React.js <br/>
- PostgreSQL <br/>
- TailwindCSS <br/>
- NextUI
